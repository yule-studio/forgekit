# Skill — positioning-message (GTM)

## When to use
새 기능 / 제품의 positioning + 핵심 메시지 작성. marketing-agent / sales-cs-agent 가 채널에 confused.

## Inputs
- 페르소나 + JTBD
- 경쟁사 / 대체재
- PRD 의 unique value

## Steps
1. **for whom**: 1-3 페르소나 + 안 맞는 사용자 명시.
2. **the problem**: 사용자가 현재 어떤 workaround / 고통.
3. **our solution**: 한 문장 + 첨예한 동사.
4. **vs alternatives**: 경쟁사 / 대체재 — 우리가 다른 점 3 개.
5. **proof point**: 사용자 quote / 데이터 / case study.
6. **call to action**: 사용자가 다음에 할 일 (signup / trial / talk).
7. **message house**: 1 메인 메시지 + 3 지지 메시지 — 채널 별 변형.

## Output
- Positioning statement (1 페이지):
  - For [target] who [need]
  - Our [product] is a [category]
  - That [unique value]
  - Unlike [alternative]
  - We [proof]
- Message house: 1 main + 3 supporting

## Quality bar
- "For whom" 명시 (대중 X)
- vs alternatives — 3 차별점 (모호 X)
- Proof point — 데이터 또는 quote
- Message house — 채널 별 (블로그 / 영업 / 광고) 변형 가능
