  ● palette modepill 표시: False (제거됨)

[D] /copy 실제 OS clipboard round-trip (pbcopy → pbpaste readback)

  /copy turn 1   → readback: 'JWT 설명\nJWT 응답'
  /copy block 1  → readback: 'JWT 설명'
  /copy all      → readback: 'JWT 설명\n\nJWT 응답\n\nTerraform\n\nTerraform 응답'

    → ForgeKit 명시 copy UX = /copy [last|turn|block|all] (위 D 가 실제 동작).
